# パスワードマネージャー (学習用)

Fletフレームワークを使用して構築された、基本的な機能を備えたGUIパスワードマネージャーです。
このプロジェクトは、セキュアなアプリケーション設計と開発の学習を目的としています。

## 主な機能

-   **マスターパスワードによる保護**: すべてのデータはマスターパスワードによって暗号化されます。
-   **安全な鍵生成**: マスターパスワードとソルトから`PBKDF2`を用いて暗号化キーを安全に導出します。
-   **強力な暗号化**: パスワードデータは`XChaCha20-Poly1305`アルゴリズムで暗号化して保存されます。
-   **パスワード生成**: 長さや使用文字（記号の有無）を指定して、ランダムなパスワードを生成できます。
-   **CRUD操作**: 保存したパスワードの追加 (Create)、表示 (Read)、編集 (Update)、削除 (Delete) が可能です。
-   **安全なクリップボード**: パスワードをクリップボードにコピーした後、一定時間で自動的にクリアする機能を備えています。

## 使い方

このアプリケーションを実行するには、[uv](https://github.com/astral-sh/uv)がインストールされている必要があります。

1.  **リポジトリをクローンします:**
    ```shell
    git clone git@github.com:VictoriaVermilion/training_password_manager.git
    cd password_manager_training
    ```

2.  **依存関係をインストールします:**
    ```shell
    uv pip install .
    ```

3.  **アプリケーションを実行します:**
    ```shell
    uv run python main.py
    ```

## ⚠️ 注意事項

-   **本番環境での使用は非推奨です**
    このアプリケーションはセキュリティの脆弱性を修正し、安全な設計原則に基づいて再構築されましたが、あくまで**学習目的**で作成されたものです。実際の重要なパスワードの管理には、[Bitwarden](https://bitwarden.com/)や[1Password](https://1password.com/)のような、専門家によって監査された信頼性の高いパスワード管理製品を使用することを強く推奨します。

-   **exeファイルのビルドについて**
    Nuitkaを使用して実行ファイル（.exe）をビルドした場合、アンチウイルスソフトによってウイルスとして誤検知される可能性があります。これは、Pythonスクリプトを単一の実行ファイルにパッケージングする際の一般的な問題です。
